# Jogo da Bolinha

## 📌 Sobre
Este projeto foi desenvolvido com o objetivo de aplicar conceitos de lógica de programação e interação básica em um jogo simples.

## 💻 Tecnologias
- Python (ou JavaScript — ajuste conforme você usou)

## 🎯 Objetivo
Praticar lógica de programação e conceitos básicos de desenvolvimento por meio da criação de um jogo simples.

## 🚀 O que foi feito
- Implementação de lógica do jogo
- Controle de movimentação
- Interação com o usuário
- Estruturação do código

## 📊 Aprendizados
- Lógica de programação
- Estruturação de código
- Resolução de problemas
- Desenvolvimento de aplicações simples
